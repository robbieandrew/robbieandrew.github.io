India's daily electricity generation

The National Load Despatch Centre of India's Power System Operation Corporation Limited (POSOCO) publishes daily reports of India's national, regional, and state-level electricity generation and demand.

These reports are in PDF format, one per day, with a lag of only one day. This web page makes those data available in machine-readable form as comma-separated value (csv) files in a single zip archive, with all data from April 2013. This is achieved by scraping the PDF files automatically to extract the relevant data.

Each report includes a number of tables, and those that are extracted here are Table A "Power Supply Position at All India and Regional level", table C "Power Supply Position in States", and table G "Sourcewise generation".

Note that table G, with a full breakdown of generation by type, was first included on 28 August 2017.

Because of persistent errors in the dating and filename formats of these files, some manual minor intervention is required each day, which means that there is no guarantee that this file will be updated daily.
