Indian energy and emissions data

These data were originally collated for preparation of the following article:

Andrew, R. 2020: "Timely estimates of India's annual and monthly fossil CO2 emissions", Earth System Science Data 12, 2411–2421, DOI: 10.5194/essd-12-2411-2020 (open access).

Since the machinery is in place, I will be continuing to update for some time. If you use the data then you should cite the original sources.

The figures here show just some examples of the data present in the CSV file.

Monthly activity data

These monthly data cover coal, oil, natural gas, cement production, and more, and are collated from many different sources as decribed in the JSON metadata file.

POSOCO: Daily electricity generation data

Generation by type and demand at regional level and demand at state level.

Data are available from 1 April 2013, although there are some gaps because of missing or unreadable reports.

CSV data file | JSON metadata file | Zip archive (both files)

CEA: Daily linkage station coal stocks

The raw data include many gaps and a number of implausible spikes. The CSV file here includes both the raw data and a processed series that fills gaps and removes spikes. For details, see my paper from 2020.
